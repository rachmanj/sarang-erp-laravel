# Training Module 8: Cross-Module Integration

## End-to-End Business Workflows

**Duration**: 3 hours  
**Target Audience**: All operational staff, managers  
**Prerequisites**: Modules 1-7, understanding of individual modules

---

## Learning Objectives

By the end of this module, participants will be able to:
1. **Understand ERP integrations** across modules for seamless workflows.
2. **Process purchase-to-pay cycle** (Purchase → Inventory → Accounting → Tax).
3. **Handle order-to-cash cycle** (Sales → Inventory → Accounting → Tax).
4. **Manage asset lifecycle integration** with financials.
5. **Use analytics integrations** for cross-module insights.
6. **Troubleshoot integration issues** in workflows.
7. **Optimize business processes** using integrated features.

---

## Module Overview

### Key Features Covered
- **Purchase-to-Pay**: PO → GRPO → Invoice → Payment → Journal/Tax.
- **Order-to-Cash**: SO → DO → Invoice → Receipt → Journal/Tax.
- **Asset Integration**: Acquisition → Depreciation → Disposal → Accounting.
- **Analytics Integration**: Data from all modules for unified reporting.
- **Error Handling**: Common integration failures and resolutions.

---

## Story-Based Training Scenarios

### Scenario 1: Purchase-to-Pay Cycle
**Business Context**: Process a complete procurement cycle from order to payment.
**Story**: "PT Sarange Trading needs new office supplies. Create a PO, receive goods, invoice, pay, and verify financial/tax impacts."

#### Step-by-Step Exploration
1. Create PO in Purchase module.
2. Process GRPO in Purchase, update Inventory.
3. Create invoice in Purchase, integrate with Tax (PPN).
4. Process payment in Accounting, post journal.
5. Verify in Reports: Check AR/AP, tax reports.

#### Discussion Points
- How does inventory update automatically?
- What tax integrations occur?

#### Hands-On Exercise
Process a full purchase-to-pay cycle with sample data.

---

### Scenario 2: Order-to-Cash Cycle
**Business Context**: Handle a customer order from sale to receipt.
**Story**: "A customer orders furniture. Process SO, delivery, invoice, receipt, and financials."

#### Step-by-Step Exploration
1. Create SO in Sales.
2. Process DO in Sales, update Inventory.
3. Create invoice in Sales, integrate with Tax.
4. Process receipt in Accounting.
5. Verify journal and reports.

#### Discussion Points
- How does credit checking integrate?
- What happens to COGS on delivery?

#### Hands-On Exercise
Complete an order-to-cash cycle.
